Dragon Fight

Language: Python

This game is something I developed during my Back End Certificate with Nucamp. In this game, the player selects one of four characters to battle the dragon, as well as a game difficulty level. The caracter then battles the dragon until there is a winner. Give this game a try to see if you can defeat the dragon!

Features of this game include:

  • 12 ways to play the game that are chosen by the player, which includes: Three difficulty levels which alter the dragon's hp and attack strength to make it more difficult to battle and four character options for the player to choose, each with unique "hp" and "attack" strength
  • The game continues to play until the player decides to exit the game
  • The player may enter either a number or a name of the character they would like to choose and is not case sensitive. Same concept for choosing a difficulty level.
  • The game keeps track of the player's character's HP remaining, as well as the dragon's and reports this throughout the game to the player and gameplay continues until the score is zero.
  • Customized game messages using string concatenation help the player understand how many times they have played the game and what is occurring in the game.

Code Features:

While loops, if/elif/else statements, conditional statements, string concatenation, infinite while loops, user selected options, variable declaration

Files:

The dragon_fight.py file on GitHub contains the entire program.

ATM App

Language: Python

The ATM app is an example of an ATM Application that could be utilized by a financial institution for withdrawing money, depositing money, or checking a deposit balance.

Features of this game include:

  • User must create a username and PIN number, which is saved in the memory.
  • Constraints on character ength of Username and PIN are enforced
  • When creating an account, there is the opportunity to enter a "promo code" to get a bonus amount of money added to the account as a promotional offer. Users have a limited number of tries to enter a promo code.
  • Once account is created, user must log in.
  • User may log-out at any time
  • Different functions defined and called upon to show balance, deposit, withdraw, and log-out.

Code Features:

While loops, if/elif/else statements, conditional statements, string concatenation, infinite while loops, user selected options, variable declaration, function declaration, packages, modules, import modules, import functions

Files:

The following files are located within the ATM app on GitHub:

  • app.py file contains the menu for the user to select a function as well as the program to call the functions for the for the different programs.
  • banking_pkg folder is a package that contains the account.py module which contains the functions for displaying the account balance, making a deposit, making a withdrawal, and logging out.

Donation App

Language: Python

The Donation Application is a program that was written to track users and the donations that they make, and also report on the total number of donations for the organization.

Features of this game include:

  • The user must log-in or register to donate. Usernames and passwords are stored in a dictionary.
  • The user must choose to login, register, donate, show donations, or exit the donation program.
  • The program verifies that the username and password are correct against the dictionary of usernames and passwords.
  • When account users donate, it tracks their donations and stores them in a dictionary.
  • When donations are shown, concatenation is used to display the individuals who donated and their donation values.
  • Users can exit the donation program, or select log-in again to log-in as a different user, or create another account.

Code Features:

While loops, if/elif/else statements, conditional statements, dictionaries, storing login info in a dictionary, string concatenation, infinite while loops, user selected options, variable declaration, function declaration, packages, modules, import modules, import functions

Files:

The following files are located within the Donation app on GitHub:

  • app.py file contains the menu for the user to select a function as well as the program to call the functions for the for the different programs.
  • donations_pkg folder is a package that contains the homepage.py module which contains definitions for functions, including a function for displaying the home screen with options, a function for donating money, and a function for showing all donations. The package also contains the user.py module which contains definitions for functions including a function to login to the donation software, as well as a function to register a username.
